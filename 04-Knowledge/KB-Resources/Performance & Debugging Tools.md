# Performance & debugging tools

---

## Browser / WebView

| Tool | Use |
|------|-----|
| **Chrome DevTools** | Performance panel, memory snapshots, Lighthouse |
| **Safari Web Inspector** | iOS WKWebView debugging |

---

## React

| Tool | Use |
|------|-----|
| **React DevTools** | Profiler, component tree |
| **why-did-you-render** (dev) | Track avoidable re-renders |

---

## Bundle

| Tool | Use |
|------|-----|
| **rollup-plugin-visualizer** | Chunk treemap |
| **vite-bundle-visualizer** | Vite-native analyze |

---

## Postgres

| Tool | Use |
|------|-----|
| **`EXPLAIN ANALYZE`** | Query plans |
| **pg_stat_statements** | Hot queries in prod (if enabled) |

---

## Mobile native

| Tool | Use |
|------|-----|
| **Android Studio Profiler** | CPU, memory |
| **Instruments** | iOS performance |
