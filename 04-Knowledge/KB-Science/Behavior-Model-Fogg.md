# Fogg behavior model

## TL;DR

- Raise **ability** (make it easier) before nagging motivation.
- **Prompt** must arrive when both sufficient.

Use with [[CPO]] for activation design.

## Nudge Theory in Product Design — 2026-04-13

- **Nudge (Thaler)** — indirect suggestions that steer decisions without restricting choice; preserves autonomy
- **Default effects** — most powerful nudge; opt-out > opt-in for desired behaviors (GDPR consent, notifications)
- **Ethical application** — nudge toward user benefit, not manipulation; disclose when using nudges
- **Product patterns** — progress bars (completion bias), social proof ("X users did this"), loss framing
