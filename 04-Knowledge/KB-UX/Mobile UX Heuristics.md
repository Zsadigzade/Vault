# Mobile UX heuristics

---

## Nielsen’s heuristics (mobile lens)

| Heuristic | Mobile twist |
|-----------|--------------|
| **Visibility of system status** | Connection, upload progress, background sync |
| **Match real world** | Platform idioms (sheet vs full page) |
| **User control & freedom** | Swipe-back, undo toast, easy dismiss |
| **Consistency** | Same iconography for actions across feeds |
| **Error prevention** | Confirm destructive; disable invalid actions |
| **Recognition > recall** | Recent searches, chips, not buried menus |
| **Flexibility** | Shortcuts for power users (long-press) |
| **Aesthetic & minimal** | One primary CTA per screen |
| **Recover from errors** | Clear message + **retry** |
| **Help & docs** | Contextual ? links, not PDF manuals |
