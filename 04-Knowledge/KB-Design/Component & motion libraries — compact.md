# Component & motion libraries — compact

> **BRUH stack:** React 18 · Tailwind · Radix (see [[Design Systems & Component Patterns]]). Use this table for **extra** libs / docs — **no API keys** for docs; npm install as needed.

| Library | URL | Use |
|---------|-----|-----|
| **shadcn/ui** | [ui.shadcn.com](https://ui.shadcn.com/) | Copy-paste Radix+Tailwind primitives · **Cursor MCP:** `npx shadcn@latest mcp init --client cursor` → [[Cursor Tips & Power Features]] · [MCP docs](https://ui.shadcn.com/docs/mcp) |
| **Radix Primitives** | [radix-ui.com](https://www.radix-ui.com/primitives) | A11y headless building blocks |
| **Lucide** | [lucide.dev](https://lucide.dev/) | Icons (already common in Vite stacks) |
| **Phosphor** | [phosphoricons.com](https://phosphoricons.com/) | Alternative icon set |
| **Framer Motion** | [motion.dev](https://motion.dev/) | Declarative motion |
| **react-spring** | [react-spring.dev](https://www.react-spring.dev/) | Physics-style animation |
| **Floating UI** | [floating-ui.com](https://floating-ui.com/) | Popovers, tooltips positioning |
| **Embla Carousel** | [embla-carousel.com](https://www.embla-carousel.com/) | Touch-friendly carousels |
| **TanStack Table** | [tanstack.com/table](https://tanstack.com/table) | Headless tables |
| **Vaul** | [github.com/emilkowalski/vaul](https://github.com/emilkowalski/vaul) | Drawer/sheet (mobile) |
| **Sonner** | [sonner.emilkowal.ski](https://sonner.emilkowal.ski/) | Toasts |
| **Fontsource** | [fontsource.org](https://fontsource.org/) | Self-host fonts, no Google CDN |
| **Uiverse** | [uiverse.io](https://uiverse.io/) | Community CSS/UI snippets — check license |
