# Agent context budget

| Layer | Max tokens (target) | Load |
|-------|---------------------|------|
| 0 | ~200 | [[HOME]] TL;DR + [[VAULT_CONSTITUTION]] skim |
| 1 | ~800 | [[SESSION_HANDOFF]] + project hub ([[BRUH_PROJECT_HOME]] if BRUH) |
| 2 | ~1000 | Active `01-Agents/roles/*.md` |
| 3 | ~2000 | ≤2 task notes from `04-Knowledge/` or `05-Projects/` |
| 4 | On demand | Deep subsystem only when editing that area |

## Note hygiene

- Long notes: **`## TL;DR`** (3 bullets) at top; agents read TL;DR before body.
- Skip by default: `07-Learning/Vault history*`, `06-Inbox/archive/**`, large binary `attachments/`.
