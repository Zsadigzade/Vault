# Agent failures log

> [!note] **Purpose:** Record agent mistakes (hallucinations, rule violations, loops). Each entry should suggest a **prevention** (vault note, rule, or test). Keep rows short.

---

## How to log

| Column | Content |
|--------|---------|
| **Date** | ISO `YYYY-MM-DD` |
| **What happened** | One or two sentences |
| **Root cause** | Wrong assumption, missed file, ignored rule, etc. |
| **Prevention** | Link to updated note, `CLAUDE.md`, Cursor rule, or test |
